TestCode_NbtControl
===================

1.6.x以前→1.7.2へ変換するためにNBTいじってた奴

- ItemSack_Fix  
  1.6.xの竹MODの袋の内部データを数字IDから内部名へ書き換えるツール
- Mod ID shifter  
  FMLがlevel.dat内で先頭からIDを詰めちゃうのが嫌だからIDを移動させるツール(個別移動やmcaの書き換えは未対応)
- NBT View Test  
  NBTの読み込みテストの残骸

※ビルドしたり動かすにはfNbt(http://www.fcraft.net/fnbt/)が必要です
